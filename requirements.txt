**Primary Users:** End users who upload Excel files and ask natural-language questions about data  
**Tech Constraints:** Python FastAPI, Next.js, PostgreSQL with pgvector extension, Docker Compose, OpenAI text-embedding-3-small, Google Generative AI SDK (gemini-2.0-flash), pandas  

**Project Summary:** This project builds a web application that allows users to upload Excel (.xlsx) files and ask natural-language questions about the data. The system ingests files by parsing them with pandas, chunking the content, embedding chunks with OpenAI, and storing vectors in pgvector. On query, it retrieves relevant chunks via cosine similarity, passes them to Google's Gemini model for answer generation with source citations, and manages user sessions with full conversation history in PostgreSQL.
